# sabcli

A Python CLI tool with various utilities.

## Installation

```bash
pip install .
sabcli echo "Hello, World!"
sabcli echo "Hello, World!" --reverse

scli echo "Hello, World!"
scli echo "Hello, World!" --reverse
```
